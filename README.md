# agent-pipeline

Reusable GitHub Actions workflows and supporting scripts that drive a personal
Issue→PR autonomous pipeline using Claude Code in GitHub Actions.

See [`docs/DESIGN.md`](docs/DESIGN.md) for architecture, [`docs/DECISIONS.md`](docs/DECISIONS.md)
for ADRs, and [`docs/CONSUMER-SETUP.md`](docs/CONSUMER-SETUP.md) for wiring this
into a consumer repo.
