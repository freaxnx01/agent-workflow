# Asking me to do things

Whenever you ask me to do something — action items, manual steps, things for me to do
outside the session — always present them as a **numbered** Markdown task checklist:

- [ ] **1.** <task 1>
- [ ] **2.** <task 2>
- [ ] **3.** <task 3>

Use one `- [ ]` line per task so each renders as a tickable checkbox, and start each
item with a bold sequential number (`**1.**`, `**2.**`, …) so I can reference them by
number in my reply (e.g. "do 1 and 3, skip 2").

Numbering restarts at 1 in each new checklist (each individual message), not across
the whole conversation — keep it local and obvious.
